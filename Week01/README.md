# Engineering Toolkit

## Week 1

### Features

- Welcome screen
- Menu system
- Greeting option
- Four-function calculator (+, -, *, /)
- Division by zero protection

### Concepts Learned

- Variables
- `int`
- `char`
- `float`
- `printf()`
- `scanf()`
- `if / else if`
- Type casting

### Build

```powershell
gcc main.c -o toolkit
```

### Run

```powershell
.\toolkit.exe
```
